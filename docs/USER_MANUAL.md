## Getting Started

### System Overview

SE Repairs is a comprehensive fleet management system designed to streamline repair operations across your organization. The system provides different interfaces for different user roles:

- **Drivers**: Report issues and track repair status
- **Workshop Staff**: Manage work orders and repair schedules
- **Operations**: Monitor fleet status and generate reports
- **Administrators**: Manage users, vehicles, and system settings

### Accessing the System

#### Web Application
1. Open your web browser
2. Navigate to your organization's SE Repairs URL
3. Enter your credentials
4. Select your preferred language (if applicable)

#### Mobile Access
- The system is fully responsive and works on mobile devices
- For the best mobile experience, add the web app to your home screen

### First-Time Login

1. **Receive Credentials**: Your system administrator will provide your login credentials
2. **Initial Login**: Use the temporary password provided
3. **Change Password**: You'll be prompted to create a new secure password
4. **Profile Setup**: Complete your profile information
5. **Tutorial**: Take the guided tour to familiarize yourself with the interface

## Driver Portal

### Reporting Issues

#### Quick Report
1. Click **"Report Downtime"** on the homepage
2. Select your vehicle from the dropdown
3. Choose the issue category
4. Describe the problem
5. Add photos or videos if possible
6. Submit the report

#### Detailed Report Process

**Step 1: Vehicle Selection**
- Select your assigned vehicle
- If your vehicle isn't listed, contact operations
- Verify vehicle details (license plate, model)

**Step 2: Issue Details**
- **Title**: Brief description (e.g., "Engine overheating")
- **Category**: Select from predefined categories
  - Engine Issues
  - Brake Problems
  - Electrical Issues
  - Transmission Problems
  - Body/Exterior Damage
  - Other
- **Priority**: System auto-assigns based on category
- **Location**: Current location (auto-detected or manual entry)

**Step 3: Description**
- Provide detailed description of the issue
- Include when the problem started
- Mention any warning lights or sounds
- Note any recent maintenance

**Step 4: Media Upload**
- Take clear photos of the problem area
- Record videos if helpful (max 2 minutes)
- Supported formats: JPG, PNG, MP4, MOV
- Maximum file size: 10MB per file

**Step 5: Review and Submit**
- Review all information for accuracy
- Submit the report
- Note your ticket number for reference

### Tracking Your Reports

#### My Tickets Dashboard
Access your tickets through:
- Homepage → "My repair tickets"
- Navigation menu → "Issues"

#### Ticket Status Meanings
- **Open**: Issue reported, awaiting assignment
- **Assigned**: Mechanic assigned, pending scheduling
- **Scheduled**: Repair appointment booked
- **In Progress**: Repair work underway
- **Waiting for Parts**: Repair paused, parts on order
- **Completed**: Repair finished, awaiting verification
- **Closed**: Issue resolved and verified

#### Ticket Details
Click on any ticket to view:
- Complete issue description
- Assigned mechanic
- Scheduled repair date/time
- Work progress updates
- Communication history
- Estimated completion time

### Communication

#### Adding Comments
1. Open your ticket
2. Scroll to the comments section
3. Type your message
4. Click "Add Comment"
5. Receive notifications when others respond

#### Receiving Updates
- Email notifications for status changes
- SMS alerts for urgent updates (if enabled)
- In-app notifications when logged in

### Emergency Procedures

#### Immediate Safety Issues
For safety-critical issues:
1. **Stop driving immediately** if unsafe
2. Call the emergency hotline: **1-800-SE-REPAIRS**
3. Report through the app with "URGENT" priority
4. Follow dispatcher instructions

#### After-Hours Reporting
- System available 24/7 for issue reporting
- Emergency issues receive immediate attention
- Non-urgent issues processed next business day

## Workshop Dashboard

### Overview

The Workshop Dashboard provides mechanics and workshop supervisors with tools to manage repair work efficiently.

### Work Order Management

#### Kanban Board View
The main dashboard displays work orders in columns:
- **Backlog**: Unscheduled work orders
- **Scheduled**: Upcoming appointments
- **In Progress**: Current work
- **Waiting**: Pending parts or approval
- **Completed**: Finished work
- **Closed**: Verified and closed

#### Managing Work Orders

**Accepting Work Orders**
1. Review work order details
2. Check parts availability
3. Estimate repair time
4. Click "Accept" to assign to yourself

**Updating Progress**
1. Open the work order
2. Update status as work progresses
3. Add progress notes
4. Upload photos of completed work
5. Record actual time spent

**Scheduling Appointments**
1. Drag work orders to calendar slots
2. Consider repair complexity and parts availability
3. Notify driver of scheduled time
4. Block calendar for estimated duration

### Parts Management

#### Checking Parts Availability
1. Open work order
2. View required parts list
3. Check inventory status
4. Order parts if needed

#### Ordering Parts
1. Click "Order Parts" in work order
2. Select supplier
3. Enter part numbers and quantities
4. Submit order
5. Update work order status to "Waiting for Parts"

### Time Tracking

#### Recording Work Time
1. Start timer when beginning work
2. Pause for breaks or other tasks
3. Stop timer when work completed
4. Review and confirm total time
5. Submit time entry

#### Labor Codes
Use appropriate labor codes:
- **DIAG**: Diagnostic time
- **REPAIR**: Actual repair work
- **PARTS**: Parts installation
- **TEST**: Testing and verification
- **ADMIN**: Administrative tasks

### Quality Control

#### Pre-Completion Checklist
Before marking work as complete:
- [ ] All repairs performed as specified
- [ ] Quality inspection completed
- [ ] Test drive performed (if applicable)
- [ ] All tools and equipment cleaned
- [ ] Work area cleaned
- [ ] Documentation updated

#### Photo Documentation
- Take before/during/after photos
- Document any additional issues found
- Show completed repair work
- Include any replaced parts

## Troubleshooting

### Common Issues

#### Login Problems
**Issue**: Cannot log in
**Solutions**:
- Check username/password spelling
- Ensure Caps Lock is off
- Clear browser cache
- Try incognito/private mode
- Contact admin for password reset

#### Slow Performance
**Issue**: System running slowly
**Solutions**:
- Check internet connection
- Close other browser tabs
- Clear browser cache
- Restart browser
- Try different browser

#### File Upload Issues
**Issue**: Cannot upload photos/videos
**Solutions**:
- Check file size (max 10MB)
- Verify file format (JPG, PNG, MP4, MOV)
- Try different file
- Check internet connection
- Use mobile app instead

#### Missing Vehicles
**Issue**: My vehicle not in dropdown
**Solutions**:
- Contact operations team
- Verify vehicle registration
- Check spelling of license plate
- Use "Other" option temporarily

### Error Messages

#### "Session Expired"
- Log out and log back in
- Clear browser cookies
- Contact admin if persistent

#### "Access Denied"
- Check user permissions
- Contact supervisor
- Verify correct role assignment

#### "Server Error"
- Refresh the page
- Try again in a few minutes
- Contact technical support

## Frequently Asked Questions

### General Questions

**Q: How do I change my password?**
A: Go to Profile → Settings → Change Password. Enter your current password and new password twice.

**Q: Can I use the system on my personal phone?**
A: Yes, the system works on any device with a web browser. For best experience, add it to your home screen.

**Q: What if I report the wrong issue?**
A: Contact operations immediately. They can update or cancel the ticket if work hasn't started.

**Q: How do I know when my vehicle is ready?**
A: You'll receive email and SMS notifications. You can also check the status in your tickets dashboard.

### Driver Questions

**Q: What if I'm not sure what's wrong with my vehicle?**
A: Describe the symptoms as best you can. Include any warning lights, sounds, or changes in performance.

**Q: Can I request a specific mechanic?**
A: You can add a note in your report, but assignment depends on availability and expertise.

**Q: What if my issue is urgent?**
A: Mark it as urgent in the report and call the emergency hotline immediately.

**Q: How long do repairs usually take?**
A: This varies by issue complexity. You'll receive an estimated completion time when work is scheduled.

### Workshop Questions

**Q: How do I handle parts that aren't in inventory?**
A: Order through the system and update the work order status to "Waiting for Parts."

**Q: What if I find additional problems during repair?**
A: Document the findings, take photos, and contact operations for approval before proceeding.

**Q: How do I handle warranty work?**
A: Follow your organization's warranty procedures and document all warranty-related work.

### Operations Questions

**Q: How do I generate monthly reports?**
A: Go to Reports → Monthly Summary, select the date range, and click Generate.

**Q: Can I export data to Excel?**
A: Yes, most reports can be exported as CSV files that open in Excel.

**Q: How do I set up automated notifications?**
A: Go to Admin → Notifications and configure the automated rules and schedules.

### Technical Questions

**Q: What browsers are supported?**
A: Chrome, Firefox, Safari, and Edge (latest versions). Internet Explorer is not supported.

**Q: Is my data secure?**
A: Yes, all data is encrypted and stored securely. The system meets industry security standards.

**Q: Can I access the system from multiple devices?**
A: Yes, you can log in from any device. Your session will remain active across devices.

**Q: What happens if the internet goes down?**
A: The mobile app has offline capabilities. Data will sync when connection is restored.
